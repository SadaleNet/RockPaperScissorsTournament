History

This tournament is originated from The International Asynchronous Rock Scissors Paper Competition in the April Fool of 2015. The original website were developed using Python with Django. No work were put onto the UI design of the website. Therefore, the website looks ugly. And it wasn't possible for the player to pick the representing nation.

In April Fool of 2016, the tournament tried to reach out more players by making this event more like an internation event. Similar to Olympic Games, we tried passing the host to different country in each year. With improved UI of the website using Semantic-UI, the event managed to get 29 players! The event was hosted by someone living in Poland. It was possible for the players to pick their representing country.

In April Fool of 2017, we've got a hoster from yet another country. Unfornately, the hoster failed to setup the website on time. Therefore, the event was cancelled.

In April Fool of 2018, Sadale took back control of the event. The frontend was pretty much the same as the 2017 one. But the backend were re-written using Flask. In addition, a few new features were implemented, including certificate of participation and a more human-friendly captcha system. The event went smoothly and everyone were happily taking the post-participation certificate.

In October 2018, Sadale had dockerized the Flask webapp and released the source code of the program to https://github.com/SadaleNet/RockPaperScissorsTournament. This event is hosted by using the webapp released to the aforementioned github link.
